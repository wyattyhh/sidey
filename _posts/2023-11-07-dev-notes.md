- css属性选择器 #css
	``` javascript
	<div class="test" :someParams="true"><div/>
	
	.test {
		&[someParams="true"] {
			color: red
		}
	}
	```

- 直接在vue页面使用方法的话每次渲染都会执行一次方法,最好用computed替代. #vue 
	``` js
	<div>{{getSum()}}</div>
	```

- Vue事件传参 #vue 
	 1. 不带圆括号时, event对象将被自动当作实惨传入. @click="onClick"  
	 2. 带圆括号时, 需要使用$event变量显式传入event对象. @click="onClick($event, val)"
